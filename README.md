# Tiles Gallery

> **Style. Strength. Satisfaction.**  
> Providing reliable tiles since 1992.
A modern tile browsing web application where users can explore elegant tile designs, view detailed product information, and manage their profile — all in a beautifully crafted UI.

🔗 **Live Demo:** [tiles-gallery-beta.vercel.app](https://tiles-gallery-beta.vercel.app/)

---

The app features a curated collection of tiles including:
-  Stone Rustic Tile Outdoor
-  Classic Brick Tile
-  Rose Gold Luxury Tile
-  Ocean Blue Mosaic

##  Features

-  **Home Page** — Hero banner with featured tile collections and latest news
-  **All Tiles** — Browse the full tile catalog with detailed views
-  **Tile Details** — Individual product pages for each tile
-  **My Profile** — User profile management
-  **Responsive Design** — Works across all screen sizes
-  **Navigation** — Smooth routing between pages
